# Portfolio Website

A minimalist portfolio website showcasing my professional experience, skills, and projects. Built with vanilla HTML, CSS, and JavaScript, inspired by Camille Mormal's design aesthetic.

🔗 **Live Demo**: [View Portfolio](https://jnaneshd.github.io/MyPortfolio/)

## ✨ Features

- **Minimalist Design**: Clean, dark theme with smooth animations
- **Drag-to-Scroll**: Interactive click-and-drag scrolling on desktop
- **Scroll Animations**: Intersection Observer-based reveal animations
- **Contact Form**: Integrated Postcatch form for direct messaging
- **Responsive**: Fully responsive design for all devices
- **Performance Optimized**: Smooth 60fps animations with hardware acceleration

## 🛠️ Technologies

- **HTML5**: Semantic markup
- **CSS3**: Custom properties, Grid, Flexbox
- **JavaScript**: Vanilla JS with modern ES6+ features
- **Postcatch**: Form submission handling

## 📂 Structure

```
Portfolio/
├── index.html          # Main HTML structure
├── style.css           # Complete design system and styles
├── script.js           # Interactive features and animations
└── README.md           # This file
```

## 🚀 Quick Start

1. Clone the repository:

```bash
git clone https://github.com/JnaneshD/MyPortfolio.git
```

2. Open `index.html` in your browser:

```bash
cd MyPortfolio
# Open index.html in your preferred browser
```

That's it! No build process or dependencies required.

## 🎨 Design Features

### Drag-to-Scroll

- Desktop-only feature (disabled on mobile for native touch scrolling)
- Click and drag anywhere to scroll vertically
- Smooth cursor transitions (grab → grabbing)

### Scroll Animations

- Fade-in animations triggered by Intersection Observer
- Staggered animations for list items
- Parallax effect on hero section

### Contact Form

- Real-time form validation
- Loading states and success/error messages
- Form resets after successful submission

## 📱 Responsive Breakpoints

- **Desktop**: 1024px and above
- **Tablet**: 768px - 1024px
- **Mobile**: Below 768px

## 🌐 Deployment

### GitHub Pages

1. Go to your repository settings
2. Navigate to Pages section
3. Select `main` branch as source
4. Your site will be live at `https://jnaneshd.github.io/MyPortfolio/`

### Netlify

1. Drag and drop the Portfolio folder to Netlify
2. Or connect your GitHub repository for automatic deployments

### Vercel

```bash
vercel --prod
```

## 📝 Customization

### Update Personal Information

Edit `index.html` to update:

- Name and title
- About section content
- Experience timeline
- Skills and projects
- Contact information

### Modify Design

Edit `style.css` to customize:

- Color scheme (CSS custom properties at top of file)
- Typography and spacing
- Animation timings
- Responsive breakpoints

### Add Features

Edit `script.js` to:

- Add new interactive features
- Modify scroll behavior
- Customize form handling
